# GDG technical writing workshop

To follow along its important you have [Node](https://nodejs.org/en/download) installed.

This workshop focuses on:

- [Vale](https://vale.sh/)
- [Docs as Test](https://doc-detective.com/docs/get-started/intro)
- [Mintlify](https://www.mintlify.com/docs)

## Use this repository

Follow the steps below to clone and open the repository:

1. Open your terminal.

2. Run the following command:

```bash
git clone https://github.com/Iheanacho-ai/gdg-tech-writing-workshop.git
```

3. Move into the project folder:

```bash
cd gdg-tech-writing-workshop
```

4. (Optional) Open the folder in your code editor:

```bash
code .
```
